Typestate Programming

Khái niệm typestate mô tả việc mã hóa thông tin về trạng thái hiện tại của một đối tượng vào kiểu của đối tượng đó. Mặc dù nghe có vẻ hơi khó hiểu, nhưng nếu bạn đã từng dùng Builder Pattern trong Rust, bạn đã bắt đầu sử dụng Typestate Programming rồi! 

pub mod foo_module {
    #[derive(Debug)]
    pub struct Foo {
        inner: u32,
    }

    pub struct FooBuilder {
        a: u32,
        b: u32,
    }

    impl FooBuilder {
        pub fn new(starter: u32) -> Self {
            Self {
                a: starter,
                b: starter,
            }
        }

        pub fn double_a(self) -> Self {
            Self {
                a: self.a * 2,
                b: self.b,
            }
        }

        pub fn into_foo(self) -> Foo {
            Foo {
                inner: self.a + self.b,
            }
        }
    }
}

fn main() {
    let x = foo_module::FooBuilder::new(10)
        .double_a()
        .into_foo();

    println!("{:#?}", x);
}

Trong ví dụ này, không có cách trực tiếp nào để tạo một đối tượng Foo. Chúng ta phải tạo một FooBuilder và khởi tạo nó đúng cách trước khi có thể lấy được đối tượng Foo mà chúng ta muốn.

Ví dụ tối giản này mã hóa hai trạng thái:

  • FooBuilder, đại diện cho trạng thái "chưa được cấu hình" hoặc "đang trong quá trình cấu hình"
  • Foo, đại diện cho trạng thái "đã được cấu hình" hoặc "sẵn sàng để dùng"

Strong Types

Vì Rust có một Strong Type System, không có cách nào dễ dàng để tạo một instance của Foo một cách "ma thuật", hoặc để biến một FooBuilder thành Foo mà không gọi phương thức into_foo(). Ngoài ra, việc gọi phương thức into_foo() tiêu thụ cấu trúc FooBuilder gốc, nghĩa là nó không thể được tái sử dụng mà không tạo một instance mới.

Điều này cho phép chúng ta biểu diễn các trạng thái của hệ thống dưới dạng các kiểu, và bao gồm các hành động cần thiết cho việc chuyển đổi trạng thái vào các phương thức trao đổi một kiểu với kiểu khác. Bằng cách tạo một FooBuilder và trao đổi nó lấy đối tượng Foo, chúng ta đã đi qua các bước của một state machine cơ bản.
