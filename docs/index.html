Rust Embedded Book

Hướng dẫn lập trình hệ thống nhúng với Rust — Phiên bản tiếng Việt

Chào mừng bạn đến với The Embedded Rust Book — tài liệu hướng dẫn sử dụng ngôn ngữ Rust cho lập trình hệ thống nhúng (embedded systems). Đây là tài liệu được dịch và chú thích sang tiếng Việt từ bản gốc tiếng Anh.
