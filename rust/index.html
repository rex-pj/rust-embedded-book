The Rust Programming Language

Cuốn sách chính thức về Rust — được dịch sang Tiếng Việt để hỗ trợ cộng đồng lập trình viên Việt Nam tiếp cận ngôn ngữ lập trình hệ thống mạnh mẽ nhất hiện nay.

Lộ trình học Rust

Giới thiệu về cuốn sách

The Rust Programming Language là cuốn sách chính thức, toàn diện nhất về Rust — viết bởi Steve Klabnik, Carol Nichols và Chris Krycho cùng sự đóng góp của cộng đồng Rust. Cuốn sách này được dịch sang Tiếng Việt nhằm giúp các lập trình viên Việt Nam dễ dàng tiếp cận Rust, ngôn ngữ kết hợp hiệu suất của C/C++ với sự an toàn bộ nhớ mà không cần garbage collector.
