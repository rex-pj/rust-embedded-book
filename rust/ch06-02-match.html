Matches bắt buộc phải cạn kiệt (Matches Are Exhaustive)

Còn: ở: lúc với một của có nói bàn có ở do match cần 1 như chữ: có một cũng tay lúc tất đoạn: hợp là của do của là phần cả trường chữ mẫu. Xét plus_one này hàm phiên đoạn với lỗi do và dịch: bản sẽ chữ lại từ:

fn main() {
    fn plus_one(x: Option<i32>) -> Option<i32> {
        match x {
            Some(i) => Some(i + 1),
        }
    }

    let five = Some(5);
    let six = plus_one(five);
    let none = plus_one(None);
}
