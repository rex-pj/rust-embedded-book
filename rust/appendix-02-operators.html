Phụ lục B: Operators và Symbols

Phụ lục này chứa bảng thuật ngữ về cú pháp của Rust, bao gồm các toán tử và các ký hiệu khác xuất hiện độc lập hoặc trong ngữ cảnh của paths, generics, trait bounds, macros, attributes, comments, tuples và brackets.

Operators

Bảng B-1 chứa các toán tử trong Rust, ví dụ về cách toán tử đó xuất hiện trong ngữ cảnh, giải thích ngắn gọn và liệu toán tử đó có thể overload được hay không. Nếu một toán tử có thể overload, trait tương ứng cần dùng để overload sẽ được liệt kê.

Bảng B-1: Operators

OperatorVí dụMô tảOverloadable?
!ident!(...), ident!{...}, ident![...]Khai triển macro
!!exprPhủ định bitwise hoặc logicNot
!=expr != exprSo sánh khác nhauPartialEq
%expr % exprChia lấy dưRem
%=var %= exprChia lấy dư và gánRemAssign
&&expr, &mut exprBorrow
&&type, &mut type, &'a type, &'a mut typeKiểu con trỏ được borrow
&expr & exprBitwise ANDBitAnd
&=var &= exprBitwise AND và gánBitAndAssign
&&expr && exprAND logic ngắn mạch
*expr * exprNhân số họcMul
*=var *= exprNhân số học và gánMulAssign
**exprDereferenceDeref
**const type, *mut typeCon trỏ thô (raw pointer)
+trait + trait, 'a + traitRàng buộc kiểu kết hợp

+=var += exprCộng số học và gánAddAssign
,expr, exprDấu phân cách tham số và phần tử

Các ký hiệu không phải toán tử

Các bảng sau chứa tất cả các ký hiệu không hoạt động như toán tử; tức là chúng không hoạt động như lời gọi hàm hay method.

Bảng B-2 hiển thị các ký hiệu xuất hiện độc lập và hợp lệ ở nhiều vị trí khác nhau.

Bảng B-2: Cú pháp độc lập

SymbolMô tả
'identLifetime hoặc nhãn vòng lặp có tên
Chữ số theo ngay sau u8, i32, f64, usize, v.v.Literal số với kiểu cụ thể
"..."String literal
r"...", r#"..."#, r##"..."##, v.v.Raw string literal; không xử lý ký tự thoát
b"..."Byte string literal; tạo mảng bytes thay vì string
br"...", br#"..."#, br##"..."##, v.v.Raw byte string literal; kết hợp raw và byte string literal
'...'Character literal
b'...'ASCII byte literal
|…| exprClosure
!Kiểu bottom luôn rỗng cho diverging functions
_Ràng buộc pattern "bị bỏ qua"; cũng dùng để làm integer literal dễ đọc hơn

Bảng B-3 hiển thị các ký hiệu xuất hiện trong ngữ cảnh path xuyên qua cấu trúc module đến một item.

Bảng B-3: Cú pháp liên quan đến Path

SymbolMô tả
ident::identĐường dẫn namespace
::pathPath tương đối với crate root (tức path tuyệt đối tường minh)
self::pathPath tương đối với module hiện tại (tức path tương đối tường minh)
super::pathPath tương đối với module cha của module hiện tại
type::ident, <type as trait>::identCác hằng, hàm và kiểu liên kết
<type>::...Item liên kết cho kiểu không thể đặt tên trực tiếp (ví dụ: <&T>::..., <[T]>::..., v.v.)
trait::method(...)Làm rõ lời gọi method bằng cách đặt tên trait định nghĩa nó
type::method(...)Làm rõ lời gọi method bằng cách đặt tên kiểu nó được định nghĩa cho
<type as trait>::method(...)Làm rõ lời gọi method bằng cách đặt tên cả trait lẫn kiểu

Bảng B-4 hiển thị các ký hiệu xuất hiện trong ngữ cảnh dùng generic type parameters.

Bảng B-4: Generics

SymbolMô tả
path<...>Chỉ định tham số cho generic type trong một kiểu (ví dụ: Vec<u8>)
path::<...>, method::<...>Chỉ định tham số cho generic type, hàm hoặc method trong một expression; thường gọi là turbofish (ví dụ: "42".parse::<i32>())
fn ident<...> ...Định nghĩa hàm generic
struct ident<...> ...Định nghĩa struct generic
enum ident<...> ...Định nghĩa enum generic
impl<...> ...Định nghĩa implementation generic
for<...> typeRàng buộc lifetime bậc cao
type<ident=type>Generic type trong đó một hoặc nhiều associated types có giá trị cụ thể (ví dụ: Iterator<Item=T>)

Bảng B-5 hiển thị các ký hiệu xuất hiện trong ngữ cảnh ràng buộc generic type parameters với trait bounds.

Bảng B-5: Ràng buộc Trait Bound

SymbolMô tả
T: UGeneric parameter T bị ràng buộc về các kiểu implement U
T: 'aGeneric type T phải sống lâu hơn lifetime 'a (nghĩa là kiểu đó không được chứa bất kỳ reference nào có lifetime ngắn hơn 'a)
T: 'staticGeneric type T không chứa borrowed reference nào ngoài các reference 'static
'b: 'aGeneric lifetime 'b phải sống lâu hơn lifetime 'a
T: ?SizedCho phép generic type parameter là dynamically sized type
'a + trait, trait + traitRàng buộc kiểu kết hợp

Bảng B-10: Dấu ngoặc vuông

Ngữ cảnhMô tả
