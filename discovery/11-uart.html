UART 🔄

MCU trên MB2 có hai biến thể UART peripheral: UART truyền thống và UARTE nâng cao (với Easy DMA). Chương này tập trung vào UARTE để giao tiếp serial phần cứng.

Khởi tạo UARTE

rust
use nrf52833_hal::uarte::{Uarte, Parity, Baudrate};

let uarte = Uarte::new(
    board.UARTE0,
    board.uart.into(),   // TX và RX pins
    Parity::EXCLUDED,    // không dùng parity bit
    Baudrate::BAUD115200,
);

Gửi một byte

Khởi tạo UARTE với baud rate đúng

Đảm bảo cả hai bên (MB2 và máy tính) đều dùng cùng baud rate — thường là 115200.

Cấu hình TX và RX pins

Trên MB2, UART pins được expose qua BSP's board.uart.

Ghi byte vào transmit buffer

rust
let byte: u8 = b'A';
uarte.write(&[byte]).unwrap();

Gửi một chuỗi

Gửi string là gửi nhiều bytes liên tiếp:

rust
let message = b"Hello from MB2!\r\n";
uarte.write(message).unwrap();
📝
Dùng \r\n (CRLF) thay vì chỉ \n

Terminal emulators thường cần cả \r (carriage return) lẫn \n (newline) để xuống dòng đúng cách — đặc biệt trên Windows với PuTTY.

Dùng macro write!

Macro write! của Rust cho phép format và gửi data — giống println! nhưng cho embedded. Cần implement trait Write cho UART type:

rust
use core::fmt::Write;

// Sau khi wrap uarte trong struct implement fmt::Write:
write!(serial, "Sensor value: {}\r\n", sensor_reading).unwrap();

Nhận một byte

Nhận data cần đọc từ receive buffer của UART. UARTE có thể báo hiệu khi có data qua polling hoặc interrupts:

rust
let mut buf = [0u8; 1];
uarte.read(&mut buf).unwrap();
let received_byte = buf[0];

Echo Server

Bài tập cổ điển với UART: xây dựng echo server — đọc bất cứ thứ gì máy tính gửi và gửi lại ngay. Điều này xác nhận cả transmit lẫn receive đều hoạt động:

rust
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use microbit::Board;
use nrf52833_hal::uarte::{Uarte, Parity, Baudrate};
use panic_rtt_target as _;

#[entry]
fn main() -> ! {
    let board = Board::take().unwrap();
    let mut serial = Uarte::new(
        board.UARTE0,
        board.uart.into(),
        Parity::EXCLUDED,
        Baudrate::BAUD115200,
    );

    let mut buf = [0u8; 1];
    loop {
        // Đọc 1 byte từ UART
        serial.read(&mut buf).unwrap();
        // Gửi lại (echo)
        serial.write(&buf).unwrap();
    }
}

Bài tập: Đảo ngược chuỗi

Bài tập nâng cao hơn: nhận chuỗi ký tự (kết thúc bằng newline), đảo ngược chuỗi đó, rồi gửi lại. Bài này luyện buffer management và string manipulation trong môi trường no-std:

Giải pháp (my solution)

Dùng fixed-size buffer để tích lũy bytes đến cho đến khi phát hiện newline, rồi đảo nội dung buffer và truyền ngược lại:

rust
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use microbit::Board;
use nrf52833_hal::uarte::{Uarte, Parity, Baudrate};
use panic_rtt_target as _;

const BUF_SIZE: usize = 32;

#[entry]
fn main() -> ! {
    let board = Board::take().unwrap();
    let mut serial = Uarte::new(
        board.UARTE0, board.uart.into(),
        Parity::EXCLUDED, Baudrate::BAUD115200,
    );

    let mut buf = [0u8; BUF_SIZE];
    let mut pos = 0;

    loop {
        let mut byte = [0u8; 1];
        serial.read(&mut byte).unwrap();

        if byte[0] == b'\n' || byte[0] == b'\r' {
            // Đảo ngược buffer và gửi
            buf[..pos].reverse();
            serial.write(&buf[..pos]).unwrap();
            serial.write(b"\r\n").unwrap();
            pos = 0; // reset buffer
        } else if pos < BUF_SIZE {
            buf[pos] = byte[0];
            pos += 1;
        }
    }
}
