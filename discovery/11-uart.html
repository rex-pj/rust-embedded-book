UART

Microcontroller có một peripheral gọi là UART, viết tắt của Universal Asynchronous Receiver/Transmitter. Peripheral này có thể được cấu hình để hoạt động với nhiều giao thức truyền thông, bao gồm giao thức serial communication.

Xuyên suốt chương này, chúng ta sẽ dùng serial communication để trao đổi thông tin giữa microcontroller và máy tính của bạn.

Có hai loại UART peripheral trên MB2: loại cũ hơn là UART và loại mới hơn là UARTE ("UART with Easy DMA"). Trong chương này, chúng ta sẽ dùng peripheral UARTE để giao tiếp serial giữa microcontroller và máy tính.

Gửi một byte đơn

Nhiệm vụ đầu tiên của chúng ta sẽ là gửi một byte đơn từ microcontroller đến máy tính qua kết nối serial.

Để làm điều đó, chúng ta sẽ dùng đoạn code sau (file examples/send-byte.rs):

#![no_main]
#![no_std]

use cortex_m::asm::wfi;
use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

use microbit::{
    hal::uarte,
    hal::uarte::{Baudrate, Parity},
};

use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    serial.write(b'X').unwrap();
    serial.flush().unwrap();

    loop {
        wfi();
    }
}

Bạn có thể nhận thấy rằng một trong những thư viện được dùng ở đây, module serial_setup, không đến từ crates.io mà được viết riêng cho dự án này. Mục đích của serial_setup là cung cấp một wrapper tiện lợi xung quanh peripheral UARTE. Nếu muốn, bạn có thể xem module này làm gì, nhưng bạn không cần phải hiểu nó để theo dõi chương này nói chung.

Tiếp theo chúng ta sẽ thảo luận về việc khởi tạo UARTE. UARTE được khởi tạo bằng đoạn code sau:

uarte::Uarte::new(
    board.UARTE0,
    board.uart.into(),
    Parity::EXCLUDED,
    Baudrate::BAUD115200,
);

Cụ thể, chúng ta lấy ownership của peripheral UARTE trong Rust (board.UARTE0) và các pin TX/RX trên board (board.uart.into()) để không ai khác có thể can thiệp vào chúng khi đang dùng. Sau đó chúng ta truyền hai tùy chọn cấu hình: baudrate và parity. Parity là cách để kiểm tra dữ liệu có bị hỏng trong quá trình truyền không — chúng ta không dùng nó ở đây nên loại trừ nó. Cuối cùng, wrap UARTE trong kiểu UartePort.

Sau khởi tạo, chúng ta gọi serial.write(b'X').unwrap() để gửi ký tự X, rồi flush() để đảm bảo byte được gửi đi ngay thay vì bị buffer lại.

Thử nghiệm

Trước khi flash, bạn nên đảm bảo đã khởi động minicom/PuTTY vì dữ liệu nhận qua serial communication không được lưu trữ hay gì cả — chúng ta phải xem trực tiếp. Khi serial monitor đã chạy, bạn có thể flash chương trình:

$ cargo embed --example send-byte

Sau khi flash xong, bạn sẽ thấy ký tự X xuất hiện trên terminal minicom/PuTTY của bạn.

Gửi một chuỗi

Bài tập tiếp theo là gửi chuỗi "The quick brown fox jumps over the lazy dog." từ microcontroller đến máy tính của bạn. Đến lượt bạn tự viết chương trình này.

Naive approach và write!

Cách gửi chuỗi ở phần trước (duyệt từng byte) hoạt động được, nhưng thiếu đi sự tiện lợi của print! như formatting tham số. Có hai cách tiếp cận:

Naive approach

Cách đơn giản nhất: duyệt từng byte và gửi từng cái một (examples/naive-send-string.rs):

rust
#![no_main]
#![no_std]

use cortex_m::asm::wfi;
use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

use microbit::hal::uarte::{self, Baudrate, Parity};

use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    for byte in b"The quick brown fox jumps over the lazy dog.\r\n".iter() {
        serial.write(*byte).unwrap();
    }
    serial.flush().unwrap();

    loop {
        wfi();
    }
}

write! và core::fmt::Write

Trait core::fmt::Write cho phép dùng bất kỳ struct nào implement nó giống như print! trong thế giới std. Vì HAL nRF đã implement core::fmt::Write cho Uart, ta có thể dùng macro write! trực tiếp (examples/send-string.rs):

rust
#![no_main]
#![no_std]

use core::fmt::Write;
use cortex_m::asm::wfi;
use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

use microbit::hal::uarte::{self, Baudrate, Parity};

use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    write!(serial, "The quick brown fox jumps over the lazy dog.\r\n").unwrap();
    serial.flush().unwrap();

    loop {
        wfi();
    }
}

Nhận một byte đơn

Bây giờ chúng ta sẽ làm chiều ngược lại: nhận dữ liệu từ máy tính vào microcontroller.

Thư viện embedded-hal cung cấp hàm read() để chờ cho đến khi có một byte và đọc nó:

#![no_main]
#![no_std]

use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::{rprintln, rtt_init_print};

#[cfg(feature = "v1")]
use microbit::{
    hal::prelude::*,
    hal::uart,
    hal::uart::{Baudrate, Parity},
};

#[cfg(feature = "v2")]
use microbit::{
    hal::prelude::*,
    hal::uarte,
    hal::uarte::{Baudrate, Parity},
};

#[cfg(feature = "v2")]
mod serial_setup;
#[cfg(feature = "v2")]
use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    #[cfg(feature = "v1")]
    let mut serial = {
        uart::Uart::new(
            board.UART0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        )
    };

    #[cfg(feature = "v2")]
    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    loop {
        let byte = serial.read().unwrap();
        rprintln!("{}", byte);
    }
}

Nếu bạn flash chương trình này và gõ ký tự trong minicom để gửi đến microcontroller, bạn sẽ chỉ thấy các con số trên RTT console vì code nhận giá trị u8 thô. Việc chuyển đổi giá trị u8 đó thành char để in ra là bài tập cho bạn tự làm.

Echo Server

Hãy kết hợp truyền và nhận vào một chương trình để viết một echo server. Echo server gửi lại cho client chính xác những gì nó nhận được — microcontroller đóng vai server, máy tính của bạn đóng vai client.

Điều này khá đơn giản để implement. Gợi ý: làm từng byte một.

Đảo ngược chuỗi

Lần này, server cần phản hồi mỗi khi client nhấn ENTER, với mỗi phản hồi trên một dòng mới. Mỗi phản hồi là chuỗi ký tự mà client gửi — nhưng bị đảo ngược.

Lời giải của tôi

#![no_main]
#![no_std]

use core::fmt::Write;
use cortex_m_rt::entry;
use heapless::Vec;
use microbit::hal::uarte::{self, Baudrate, Parity};
use panic_rtt_target as _;
use rtt_target::rtt_init_print;
use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    // A buffer with 32 bytes of capacity
    let mut buffer: Vec<u8, 32> = Vec::new();

    loop {
        buffer.clear();

        loop {
            // We assume that the receiving cannot fail
            let byte = serial.read().unwrap();

            if buffer.push(byte).is_err() {
                write!(serial, "error: buffer full\r\n").unwrap();
                break;
            }

            if byte == b'\r' {
                for byte in buffer.iter().rev().chain(b"\r\n") {
                    serial.write(*byte).unwrap();
                }
                break;
            }
        }
        serial.flush().unwrap()
    }
}
