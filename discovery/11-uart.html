UART

Microcontroller của chúng ta (như hầu hết các MCU khác) có các UART ("Universal Asynchronous Receiver/Transmitter") peripheral. Có hai loại UART peripheral trên MB2: UART cũ hơn và UARTE mới hơn ("UART with Easy DMA").

Trong chương này, chúng ta sẽ dùng peripheral UARTE để giao tiếp với cổng serial phần cứng. Xuyên suốt chương này, chúng ta sẽ dùng serial communication để trao đổi thông tin giữa microcontroller và máy tính của bạn.

Gửi một byte

rust
#![no_main]
#![no_std]

use cortex_m::asm::wfi;
use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

use microbit::{
    hal::uarte,
    hal::uarte::{Baudrate, Parity},
};

use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    serial.write(b'X').unwrap();
    serial.flush().unwrap();

    loop {
        wfi();
    }
}

Gửi một chuỗi

Hãy gửi chuỗi "The quick brown fox jumps over the lazy dog." từ microcontroller đến máy tính của bạn.

📝
Lưu ý

Dữ liệu chúng ta nhận qua serial communication không được lưu lại: chúng ta phải xem trực tiếp khi nó đến. Hãy mở minicom (Linux/macOS) hoặc PuTTY (Windows) trước khi flash chương trình.

Nhận một byte

rust
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use panic_rtt_target as _;
use rtt_target::{rprintln, rtt_init_print};

use microbit::hal::uarte::{self, Baudrate, Parity};

use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    loop {
        let byte = serial.read().unwrap();
        rprintln!("{}", byte);
    }
}

Ở đây chúng ta dùng hàm serial.read() để chờ cho đến khi có một byte và đọc nó. Dữ liệu nhận được được in ra dưới dạng số trừ khi bạn tự chuyển đổi.

Echo Server

Hãy gộp truyền và nhận vào một chương trình và viết một echo server. Echo server gửi lại cho client chính xác những gì nó nhận được.

Gợi ý: triển khai từng byte một. Đây là bài tập — không có giải pháp mẫu được cung cấp cho phần này.

Đảo ngược chuỗi

Lần này, server cần phản hồi mỗi khi client nhấn ENTER, với mỗi phản hồi trên một dòng mới. Mỗi phản hồi là chuỗi ký tự mà client gửi — nhưng bị đảo ngược.

Giải pháp

rust
#![no_main]
#![no_std]

use core::fmt::Write;
use cortex_m_rt::entry;
use heapless::Vec;
use microbit::hal::uarte::{self, Baudrate, Parity};
use panic_rtt_target as _;
use rtt_target::rtt_init_print;
use serial_setup::UartePort;

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let mut serial = {
        let serial = uarte::Uarte::new(
            board.UARTE0,
            board.uart.into(),
            Parity::EXCLUDED,
            Baudrate::BAUD115200,
        );
        UartePort::new(serial)
    };

    // Buffer với dung lượng 32 bytes
    let mut buffer: Vec<u8, 32> = Vec::new();

    loop {
        buffer.clear();

        loop {
            // Giả sử việc nhận không bao giờ thất bại
            let byte = serial.read().unwrap();

            if buffer.push(byte).is_err() {
                write!(serial, "error: buffer full\r\n").unwrap();
                break;
            }

            if byte == b'\r' {
                for byte in buffer.iter().rev().chain(b"\r\n") {
                    serial.write(*byte).unwrap();
                }
                break;
            }
        }
        serial.flush().unwrap()
    }
}
