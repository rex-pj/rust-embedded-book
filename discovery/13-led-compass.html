LED Compass

Trong phần này, chúng ta sẽ triển khai một la bàn dùng các LED trên MB2. Như những chiếc la bàn thực sự, la bàn LED của chúng ta phải chỉ về hướng bắc bằng cách nào đó. Nó sẽ làm điều đó bằng cách bật một trong các LED ở rìa ngoài; LED được bật phải chỉ về phía bắc.

Đo từ trường

Từ trường có cả độ lớn, đo bằng Gauss hoặc Tesla, và hướng. Magnetometer trên MB2 đo cả độ lớn lẫn hướng của từ trường bên ngoài, nhưng nó báo cáo lại thành phần của trường đó theo các trục của nó.

Khi board được giữ phẳng với LED hướng lên và logo hướng về phía trước, trục X và Y tạo thành mặt phẳng của sàn. Trục X hướng về cạnh trái của board. Trục Y hướng về cạnh dưới (card connector) của board. Trục Z hướng "vào sàn", tức là hướng xuống.

⚠️
Chú ý về phần cứng

Trong số hai chiếc MB2 tác giả có lúc viết cuốn sách này, một chiếc dường như có magnetometer hơi hỏng: trục Z bị offset đến mức không dùng được.

📝
Từ bắc địa lý và từ bắc từ tính

Từ bắc của Trái Đất là thứ thất thường: nó khác với từ bắc thực sự ở hầu hết các nơi trên Trái Đất, đôi khi đáng kể. Nó có thể chỉ xuống đất khá nhiều. Ở vị trí của tác giả, "độ lệch" (sự khác biệt giữa từ bắc thực và từ bắc từ tính) là khoảng 15°; "độ nghiêng" là một con số đáng ngạc nhiên là 67° hướng xuống đất. Bạn có thể tính toán những giá trị này cho vị trí của mình bằng các máy tính trực tuyến như máy tính của US NOAA hoặc UK BGS.

Crate lsm303agr hiện tại không hỗ trợ bù đắp những điều này, và có vẻ như đây là quá nhiều cho một hướng dẫn nhập môn về hệ thống nhúng.

📝
Độ chính xác của thiết bị

Magnetometer LSM303AGR không phải là thiết bị đặc biệt chính xác khi dùng ngay từ hộp. Nhà sản xuất khuyến nghị một quy trình hiệu chỉnh phức tạp để tìm các điều chỉnh cho số đọc từ magnetometer.

Bài tập: Quan sát từ trường

Trong chương I2C, bạn đã viết chương trình examples/show-mag.rs in liên tục dữ liệu magnetometer. Dùng chương trình đó để quan sát sự thay đổi của các giá trị khi bạn xoay board theo các hướng khác nhau.

Giải pháp

rust
#![deny(unsafe_code)]
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use embedded_hal::delay::DelayNs;
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

use core::f32::consts::PI;
use libm::{atan2f, floorf};

use microbit::{
    display::blocking::Display,
    hal::{twim, Timer},
    pac::twim0::frequency::FREQUENCY_A,
};

use lsm303agr::{Lsm303agr, MagMode, MagOutputDataRate};

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = microbit::Board::take().unwrap();

    let i2c = { twim::Twim::new(board.TWIM0, board.i2c_internal.into(), FREQUENCY_A::K100) };

    let mut timer0 = Timer::new(board.TIMER0);
    let mut display = Display::new(board.display_pins);

    let mut sensor = Lsm303agr::new_with_i2c(i2c);
    sensor.init().unwrap();
    sensor
        .set_mag_mode_and_odr(
            &mut timer0,
            MagMode::HighResolution,
            MagOutputDataRate::Hz10,
        )
        .unwrap();
    let mut sensor = sensor.into_mag_continuous().ok().unwrap();

    let mut leds = [[0u8; 5]; 5];

    #[rustfmt::skip]
    let indices = [
        (2, 0), /* W */
        (3, 0), /* W-SW */
        (3, 1), /* SW */
        (4, 1), /* S-SW */
        (4, 2), /* S */
        (4, 3), /* S-SE */
        (3, 3), /* SE */
        (3, 4), /* E-SE */
        (2, 4), /* E */
        (1, 4), /* E-NE */
        (1, 3), /* NE */
        (0, 3), /* N-NE */
        (0, 2), /* N */
        (0, 1), /* N-NW */
        (1, 1), /* NW */
        (1, 0), /* W-NW */
    ];

    loop {
        while !sensor.mag_status().unwrap().xyz_new_data() {
            timer0.delay_ms(1u32);
        }
        let (x, y, _) = sensor.magnetic_field().unwrap().xyz_nt();

        let theta = atan2f(y as f32, x as f32);
        let seg = floorf(16.0 * theta / PI) as i8;

        let index = if seg >= 15 || seg <= -15 {
            8
        } else if seg >= 0 {
            (seg / 2) as usize
        } else {
            ((31 + seg) / 2) as usize
        };

        let (r, c) = indices[index];
        leds[r][c] = 255u8;
        display.show(&mut timer0, leds, 50);
        leds[r][c] = 0u8;
        display.show(&mut timer0, leds, 50);
    }
}
