LED Compass 🧭

Trong chương này, chúng ta xây dựng một la bàn LED: dùng magnetometer trên MB2 để xác định hướng từ bắc và hiển thị bằng LED trên ma trận 5×5.

Đo từ trường

Magnetometer đo cả độ lớn lẫn hướng của từ trường, báo cáo theo ba trục. Khi board nằm phẳng với LED hướng lên và logo hướng về phía trước:

📐 Hệ trục tọa độ của MB2
←
Trục X
Hướng sang cạnh trái
↓
Trục Y
Hướng về cạnh dưới (card connector)
⬇
Trục Z
Hướng xuống sàn (right-hand system)

Bài tập thực hành: Quan sát từ trường

Viết chương trình (examples/show-mag.rs) liên tục in dữ liệu magnetometer lên RTT console:

Xác định hướng Bắc tại vị trí của bạn

Dùng la bàn điện thoại để biết hướng. Căn thẳng MB2 theo hướng đó và ghi lại giá trị X, Y.

Xoay 90° và ghi lại

Giữ board song song với mặt đất (không nghiêng), xoay 90° và xem giá trị thay đổi ra sao.

Xoay thêm 90° nữa

Tiếp tục quan sát. Mỗi 90° xoay, giá trị X và Y sẽ hoán đổi và đổi dấu.

⚠️
Lưu ý về độ chính xác
  • Từ bắc địa lý ≠ từ bắc từ tính — có sai lệch (magnetic declination) tùy địa điểm
  • Một số MB2 có offset trên trục Z — cần calibration
  • Vật liệu sắt thép gần board sẽ làm nhiễu phép đo

Tính toán góc hướng (Heading)

Để xác định hướng la bàn, chỉ cần X và Y (giả sử board nằm phẳng). Dùng hàm atan2:

📐 Công thức tính heading
rust
use libm::atan2f;
use core::f32::consts::PI;

// Đọc magnetometer
let mag = sensor.magnetic_field().unwrap();
let x = mag.x_nt() as f32;
let y = mag.y_nt() as f32;

// Tính góc hướng (radians)
let heading_rad = atan2f(y, x);

// Chuyển sang degrees (0–360)
let mut heading_deg = heading_rad * 180.0 / PI;
if heading_deg < 0.0 {
    heading_deg += 360.0;
}

Ánh xạ góc → LED

Ma trận 5×5 có 8 vị trí ở rìa ngoài tương ứng với 8 hướng la bàn (N, NE, E, SE, S, SW, W, NW). Mỗi hướng chiếm ~45°:

🗺️ Ánh xạ Heading → LED Position
rust
// 8 vị trí LED rìa ngoài theo chiều kim đồng hồ từ hướng Bắc
// (row, col) trong ma trận 5x5
const COMPASS_LEDS: [(usize, usize); 8] = [
    (0, 2), // N  — góc trên giữa
    (0, 4), // NE — góc trên phải
    (2, 4), // E  — giữa phải
    (4, 4), // SE — góc dưới phải
    (4, 2), // S  — góc dưới giữa
    (4, 0), // SW — góc dưới trái
    (2, 0), // W  — giữa trái
    (0, 0), // NW — góc trên trái
];

fn heading_to_led_index(heading_deg: f32) -> usize {
    // Mỗi sector = 45°, offset 22.5° để center
    let index = ((heading_deg + 22.5) / 45.0) as usize % 8;
    index
}

Giải pháp hoàn chỉnh

rust
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use libm::atan2f;
use core::f32::consts::PI;
use lsm303agr::{MagMode, MagOutputDataRate, Lsm303agr};
use microbit::{board::Board, display::blocking::Display, hal::Timer};
use panic_rtt_target as _;
use rtt_target::rtt_init_print;

const COMPASS_LEDS: [(usize, usize); 8] = [
    (0, 2), (0, 4), (2, 4), (4, 4),
    (4, 2), (4, 0), (2, 0), (0, 0),
];

#[entry]
fn main() -> ! {
    rtt_init_print!();
    let board = Board::take().unwrap();
    let mut timer = Timer::new(board.TIMER0);
    let mut display = Display::new(board.display_pins);

    let i2c = microbit::hal::Twim::new(
        board.TWIM0, board.i2c_internal.into(), microbit::hal::twim::Frequency::K100,
    );
    let mut sensor = Lsm303agr::new_with_i2c(i2c);
    sensor.init().unwrap();
    sensor.set_mag_mode_and_odr(&mut timer, MagMode::HighResolution, MagOutputDataRate::Hz10).unwrap();

    loop {
        if sensor.mag_status().unwrap().xyz_new_data() {
            let mag = sensor.magnetic_field().unwrap();
            let x = mag.x_nt() as f32;
            let y = mag.y_nt() as f32;

            let heading_rad = atan2f(y, x);
            let mut heading_deg = heading_rad * 180.0 / PI;
            if heading_deg < 0.0 { heading_deg += 360.0; }

            let idx = ((heading_deg + 22.5) / 45.0) as usize % 8;
            let (row, col) = COMPASS_LEDS[idx];

            let mut grid = [[0u8; 5]; 5];
            grid[row][col] = 1;
            display.show(&mut timer, grid, 100);
        }
    }
}
