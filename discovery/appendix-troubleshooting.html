⚠️ Vấn đề với cargo-embed

Hầu hết các vấn đề với cargo-embed trên Linux đều liên quan đến việc chưa cài đặt đúng các quy tắc udev. Nếu gặp lỗi, bạn có thể tìm trợ giúp bằng cách:

  • Mở một issue trên repository GitHub của cuốn sách
  • Truy cập kênh Matrix của Rust Embedded
  • Truy cập kênh Matrix của probe-rs

⚠️ Vấn đề với Cargo

"can't find crate for core"

Vấn đề: Compiler không tìm thấy crate core khi biên dịch các dependency embedded.

Nguyên nhân: Chưa cài đặt target microcontroller phù hợp.

Giải pháp: Cài đặt target đúng bằng rustup:

bash
$ rustup target add thumbv7em-none-eabihf

Không thể flash: "No loadable segments were found in the ELF file"

Vấn đề: Flash thất bại với thông báo lỗi "No loadable segments were found in the ELF file".

Nguyên nhân: Cargo cần cấu hình build và link đúng trong .cargo/config.toml để phù hợp với thiết bị đích.

Giải pháp: Tạo file .cargo/config.toml chỉ định target, runner và linker flags. Ví dụ cấu hình cho nRF52833:

toml
[build]
target = "thumbv7em-none-eabihf"

[target.thumbv7em-none-eabihf]
runner = "probe-rs run --chip nRF52833_xxAA"
rustflags = [
  "-C", "linker=rust-lld",
]
