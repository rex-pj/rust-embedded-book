Snake Game 🐍

Chương này xây dựng game rắn săn mồi cổ điển trên MB2 — dùng ma trận LED 5×5 làm màn hình và hai nút bấm làm điều khiển. Đây là dự án tổng hợp toàn bộ kiến thức đã học.

Controls — Input qua Interrupt

rust
// src/controls.rs
use core::sync::atomic::{AtomicU8, Ordering};
use nrf52833_pac::interrupt;

// 0=Up, 1=Right, 2=Down, 3=Left
static DIRECTION: AtomicU8 = AtomicU8::new(0);

#[interrupt]
fn GPIOTE() {
    let gpiote = unsafe { &*nrf52833_pac::GPIOTE::ptr() };
    let dir = DIRECTION.load(Ordering::SeqCst);

    if gpiote.events_in[0].read().bits() != 0 {
        // Button A: rẽ trái (relative)
        gpiote.events_in[0].write(|w| w);
        DIRECTION.store((dir + 3) % 4, Ordering::SeqCst);
    }
    if gpiote.events_in[1].read().bits() != 0 {
        // Button B: rẽ phải (relative)
        gpiote.events_in[1].write(|w| w);
        DIRECTION.store((dir + 1) % 4, Ordering::SeqCst);
    }
}

pub fn current_direction() -> u8 {
    DIRECTION.load(Ordering::SeqCst)
}

Non-blocking Display

Ma trận 5×5 là multiplexed display: để hiển thị ảnh ổn định, các hàng phải được quét tuần tự rất nhanh. Module display dùng timer interrupt để quét tự động:

⚡ Tại sao cần Non-blocking Display?
ComponentYêu cầu tốc độGiải pháp
Game loop~2 lần/giây (500ms/step)Chạy trong main loop
LED refresh~100Hz (10ms/frame)Timer interrupt tự động
Button inputPhản hồi ngay lập tứcGPIOTE interrupt

Nếu dùng blocking display trong game loop, LED sẽ chớp nháy vì game loop quá chậm.

Final Assembly — Ghép tất cả lại

rust
#![no_main]
#![no_std]

use cortex_m_rt::entry;
use microbit::Board;
use panic_rtt_target as _;

mod game;
mod controls;
mod display;

#[entry]
fn main() -> ! {
    let board = Board::take().unwrap();

    // 1. Khởi tạo display interrupt (refresh LED ở 100Hz)
    display::init(board.display_pins, board.TIMER0);

    // 2. Khởi tạo controls interrupt (phản hồi button)
    controls::init(board.buttons, board.GPIOTE);

    // 3. Khởi tạo game state
    let mut game_state = game::GameState::new();

    loop {
        // Lấy hướng hiện tại từ controls module
        let direction = controls::current_direction();

        // Cập nhật game state
        game_state.step(direction);

        if game_state.is_over() {
            // Hiển thị animation game over
            display::show_game_over();
            // Reset game
            game_state = game::GameState::new();
        } else {
            // Cập nhật LED matrix buffer
            display::update(game_state.led_matrix());
        }

        // Ngủ đến timer interrupt tiếp theo (game tick ~500ms)
        cortex_m::asm::wfi();
    }
}
