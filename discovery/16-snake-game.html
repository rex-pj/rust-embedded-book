Snake Game

Bây giờ chúng ta sẽ triển khai một game rắn cơ bản mà bạn có thể chơi trên MB2 bằng ma trận LED 5×5 làm màn hình và hai nút bấm làm điều khiển.

Chúng ta sẽ xây dựng code theo kiểu "bottom-up" với ba module chính: game, controls và display. Mỗi module bao gồm một file nguồn cấp cao nhất cùng các subfile được include bằng câu lệnh mod của Rust.
