🌟 Ví von: Phiên dịch viên

Bạn nói tiếng Việt, bạn của bạn nói tiếng Nhật. Hai bạn vẫn có thể làm việc cùng nhau nếu có phiên dịch viên!

FFI (Foreign Function Interface) là phiên dịch viên giữa Rust và C — hai ngôn ngữ nói chuyện với nhau qua cầu nối này! 🌉

FFI (Foreign Function Interface) là "cầu nối" giúp Rust và C giao tiếp với nhau!

🌉 FFI — Cầu Nối Giữa Hai Thế Giới

FFI = Foreign Function Interface = "Giao diện hàm nước ngoài"

Cho phép:

  • 🦀→🔵 Rust gọi hàm C — dùng thư viện C trong project Rust
  • 🔵→🦀 C gọi hàm Rust — nhúng code Rust vào project C

Cả hai chiều đều hoạt động! 🔄

🖥️ RTOS — Hệ Điều Hành Nhúng Nhỏ

🏃 Khi robot cần làm nhiều việc thực sự song song, người ta dùng RTOS (Real-Time Operating System) — hệ điều hành siêu nhỏ dành cho robot!

FreeRTOS là RTOS phổ biến nhất — viết bằng C. Với FFI, bạn có thể dùng FreeRTOS từ code Rust! 🤖

  • ⏱️ Quản lý nhiều "task" chạy song song
  • 📦 Hàng đợi tin nhắn giữa các task
  • 🔐 Semaphore, Mutex cho đồng bộ

🔑 ABI (Application Binary Interface) — "Ngôn ngữ nhị phân chung"!

Cả Rust và C đều hiểu ABI chung này. Khi Rust dùng extern "C", nó nói: "Tôi sẽ nói chuyện theo kiểu C"!

  • 📐 Cách sắp xếp tham số trên stack
  • 🔢 Cách trả về giá trị
  • 📛 Cách đặt tên hàm trong binary

Miễn là cả hai nói cùng "ABI", chúng hiểu nhau! 🌍
