🦀 Đưa Rust Vào Dự Án C

Viết một phần bằng Rust, phần còn lại vẫn là C

🏠 Trang chủ › Tương tác ngôn ngữ › Rust trong C

🌱 Câu Chuyện Thực Tế

🏢 Công ty bạn có một dự án firmware lớn viết bằng C — hàng chục nghìn dòng code, đã hoạt động nhiều năm. Không thể viết lại hết từ đầu!

Nhưng bạn muốn dùng Rust cho phần mới — an toàn hơn, ít bug hơn. Giải pháp: viết phần mới bằng Rust, phần cũ vẫn là C! 🤝

🌟 Ví von: Người nước ngoài học tiếng Việt

Một chuyên gia người Anh làm việc với đội ngũ Việt Nam. Thay vì yêu cầu tất cả học tiếng Anh, chuyên gia đó học thêm tiếng Việt để giao tiếp!

Code Rust "học cách nói tiếng C" để làm việc cùng codebase C hiện tại! 🌍

📦 Tạo Thư Viện Rust cho C

Trong Cargo.toml, khai báo kiểu thư viện:

  • [lib]
  • crate-type = ["staticlib"] — thư viện tĩnh (.a file), link vào C
  • Hoặc: crate-type = ["cdylib"] — thư viện động (.so/.dll), load lúc chạy

Kết quả: một file libmyrust.a mà code C có thể link vào! 🔗

📛 #[no_mangle] — Giữ Nguyên Tên Hàm

🌟 Ví von: Mã hóa tên

Rust bình thường "mã hóa" tên hàm thành chuỗi ký tự lạ: compute_temperature → _ZN7mylib19compute_temperatureE! 😱

C không hiểu tên lạ đó! #[no_mangle] nói với Rust: "Giữ nguyên tên gốc, đừng mã hóa!" 📛

Ví dụ hàm Rust xuất sang C:

  • #[no_mangle] — giữ tên nguyên
  • pub extern "C" — xuất theo chuẩn C
  • fn compute_temperature(raw: u16) -> f32 — hàm bình thường

Bây giờ code C có thể gọi: float t = compute_temperature(raw_val); 🎉

📄 cbindgen — Tạo Header C Tự Động

🤖 cbindgen là "máy dịch ngược" — đọc code Rust và tạo file header C (.h) tự động!

Thay vì tự viết:

/* sensor.h */
float compute_temperature(uint16_t raw);
void init_sensor(const Config* config);

Chạy: cbindgen --output sensor.h → cbindgen tự làm! 🎊

🔄 Chiến Lược Chuyển Đổi Dần Dần

1️⃣
Bắt đầu nhỏ
Viết một module nhỏ bằng Rust đầu tiên — ví dụ: thuật toán tính toán không có side effect 🌱
2️⃣
Test kỹ
Đảm bảo phần Rust hoạt động đúng, unit test đầy đủ! Rust giúp viết test dễ hơn 🧪
3️⃣
Dần dần mở rộng
Viết thêm module mới bằng Rust, tái cấu trúc module C cũ khi cần sửa chữa 📈
4️⃣
Không cần làm hết
Không nhất thiết phải chuyển toàn bộ sang Rust! Kết hợp là hoàn toàn chấp nhận được 🎯

🚀 Nhiều công ty thành công làm điều này!

  • Mozilla Firefox: dần dần thêm Rust vào codebase C/C++
  • Linux kernel: mới bắt đầu cho phép Rust module
  • Android OS: dùng Rust cho các component mới

Bạn không cần làm một mình — cả thế giới đang đi con đường này! 🌍

💡 Lời khuyên: Bắt đầu bằng các phần code tự chứa (pure functions, không phụ thuộc vào global state) — dễ nhất để chuyển sang Rust và cũng dễ test nhất! 🎯
