🎯 Từ QEMU lên Robot Thật!

🤖 Bạn đã luyện tập trên robot ảo QEMU rồi! Giờ là lúc chạy code trên robot thật! Hồi hộp chưa? 🎉

Sự khác biệt lớn nhất: với robot thật, bạn phải biết chính xác thông số phần cứng của mình. Mỗi robot có cấu hình khác nhau!

📋 4 Thông Tin Cần Biết Về Robot Của Bạn

🧠
Loại CPU
Cortex-M0? M3? M4? M7? Mỗi loại có tập lệnh khác nhau → target khác nhau!
🔢
Có FPU không?
FPU = tính số thập phân nhanh. Cortex-M4F có FPU (chữ F ở cuối!) 🏃
📔
Flash bao nhiêu?
STM32F303: 256KB Flash, bắt đầu từ địa chỉ 0x08000000
🖊️
RAM bao nhiêu?
STM32F303: 40KB RAM, bắt đầu từ địa chỉ 0x20000000

⚙️ Cấu hình cho STM32F3DISCOVERY

Thông số Giá trị Ý nghĩa

🔌 Kết Nối Robot Với Máy Tính

🌟 Ví von: Dây cáp đặc biệt

ST-LINK là "dây cáp đặc biệt" tích hợp trên board. Khi bạn cắm USB vào máy tính, máy tính có thể:

  • ⚡ Flash code — Ghi chương trình vào sổ tay robot
  • 🔍 Debug — Nhìn xem robot đang làm gì bên trong
  • 🛑 Dừng/tiếp tục — Điều khiển robot từ xa

🚀 Quy Trình Flash Code Lên Robot

  1. 🔨 Build: cargo build --release — Tạo file binary tối ưu
  2. ⚡ Flash: cargo flash --chip STM32F303VCTx — Ghi vào sổ tay robot
  3. 🔄 Robot tự khởi động lại — Và tự động chạy code mới!

Hoặc dùng probe-rs để vừa flash vừa debug cùng lúc! 🛠️

⚠️ Quan trọng về điện áp! STM32F3DISCOVERY chỉ chịu được 3.3V trên các chân GPIO. Nếu bạn kết nối thiết bị 5V trực tiếp, chip sẽ bị hỏng ngay! Luôn kiểm tra datasheet trước khi kết nối bất kỳ thiết bị ngoài nào. 🛑
