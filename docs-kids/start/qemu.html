🎮 QEMU là gì?

🕹️ Bạn có biết trình giả lập game (emulator) không? Nó cho bạn chơi game Super Mario trên máy tính dù không có máy Nintendo! QEMU cũng như vậy — nhưng giả lập cả một con chip ARM thật sự!

Bạn có thể viết và chạy code robot ngay trên máy tính, không cần mua board mạch nào cả! 🎉

QEMU là "robot ảo" chạy ngay trên máy tính — luyện tập thoải mái mà không sợ làm hỏng robot thật!

🌟 Ví von

QEMU giống như phòng luyện tập ảo. Phi công tập lái máy bay trong buồng lái giả lập trước khi lái máy bay thật. Bạn cũng tập lập trình robot trên QEMU trước khi dùng robot thật! ✈️

📁 Tạo Project Đầu Tiên

Bước 1: Tạo từ template

Dùng cargo-generate — như dùng khuôn bánh 🍪:

cargo generate --git https://github.com/rust-embedded/cortex-m-quickstart

Bước 2: Đặt tên project

Nhập tên của bạn, ví dụ: hello-robot

Bước 3: Vào thư mục và chạy!

cd hello-robot
cargo run --target thumbv7em-none-eabihf

🗂️ Cấu trúc project

Project robot có những file đặc biệt:

  • 📄 src/main.rs — Code chính của bạn viết ở đây
  • 📄 memory.x — "Bản đồ bộ nhớ robot" (quan trọng!)
  • 📄 Cargo.toml — Danh sách công cụ cần dùng
  • 📄 .cargo/config.toml — Cài đặt cho Cargo và QEMU
  • 📄 build.rs — Script chạy trước khi build

🗺️ memory.x — Bản Đồ Bộ Nhớ Robot

🌟 Ví von: Bản đồ ngôi nhà robot

Giống như bản đồ ngôi nhà cho biết "phòng ngủ ở đây, nhà bếp ở đó", memory.x nói với robot: "Flash (sổ tay) bắt đầu từ địa chỉ X, RAM (bảng trắng) bắt đầu từ địa chỉ Y"!

File memory.x cho QEMU LM3S6965EVB trông như thế này:

  • 📔 Flash: Bắt đầu từ 0x00000000, dung lượng 256KB
  • 🖊️ RAM: Bắt đầu từ 0x20000000, dung lượng 64KB

👋 Hello World đầu tiên!

Chương trình đơn giản nhất — robot in "Hello, world!" ra màn hình:

  • 🔧 Dùng cortex_m_semihosting::hprintln! để in chữ
  • ⏹️ Dùng debug::exit() để thoát khi chạy trên QEMU
  • 🔄 Dùng loop {} để robot chạy mãi mãi trên phần cứng thật

Khi chạy cargo run, bạn sẽ thấy dòng chữ "Hello, world!" hiện ra! 🎊

💡 Thú vị chưa? QEMU giả lập cả CPU ARM thật! Code chạy trên QEMU giống hệt như chạy trên robot thật — chỉ không có đèn LED nhấp nháy thôi! 😄
⚠️ Nhớ nhé: debug::exit() CHỈ dùng với QEMU! Nếu bạn dùng trên robot thật, robot sẽ bị "đóng băng" không hoạt động được nữa. Sẽ học cách khác ở trang tiếp theo! 🛑
🤖 Pseudo-code vui vẻ — Chạy trên QEMU
// Chạy chương trình trên robot ảo:
cargo run --target thumbv7m-none-eabi

// Robot ảo QEMU in ra:
Hello, robot ảo! 🤖
SysTick! 1
SysTick! 2
SysTick! 3...
