⚙️ 6 Bước Cài Đặt Môi Trường

🛠️ Chuẩn bị "xưởng chế tạo robot"
  1. 🦀 Cài Rust — Vào rustup.rs, làm theo hướng dẫn
  2. 🎯 Thêm ARM target — rustup target add thumbv7em-none-eabihf
  3. 🎮 Cài QEMU — Robot ảo để luyện tập
  4. 🍪 Cài cargo-generate — Tạo project từ template
  5. 🔍 Cài GDB hoặc probe-rs — Công cụ debug
  6. 📡 Cài ST-Link (nếu có robot thật) — Driver kết nối
💡 Mẹo: Bắt đầu với QEMU — không cần mua robot! Sau khi code chạy tốt trên QEMU, mới cần robot thật! 🎉
