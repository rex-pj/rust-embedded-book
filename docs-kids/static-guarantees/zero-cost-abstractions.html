🆓 "Zero-cost" nghĩa là gì?

🎯 Zero-cost abstraction = thêm tính năng tiện lợi, an toàn hơn... nhưng robot không cần tốn thêm bộ nhớ, không chạy chậm hơn!

Đây là câu châm ngôn nổi tiếng của Rust: "What you don't use, you don't pay for. What you do use, you couldn't hand code any better."

Tạm dịch: "Thứ gì bạn không dùng thì không tốn tiền. Thứ gì bạn dùng thì code tay cũng không nhanh hơn được!" 🏆

Rust kiểm tra lúc viết code — khi chạy thật, code nhỏ gọn và nhanh như C!

🏷️ Zero Sized Types (ZST) — Nhãn Dán Vô Hình

🌟 Ví von: Nhãn dán lên hộp

Khi bạn dán nhãn "ĐỒ GIÒN" lên hộp — nhãn giúp bạn biết cần cẩn thận, nhưng nhãn không chiếm không gian trong hộp!

Zero Sized Types cũng vậy: chúng giúp compiler "biết thêm thông tin" nhưng không chiếm bất kỳ byte nào trong bộ nhớ robot! 🎁

Ví dụ: Các kiểu trạng thái cho GPIO:

  • struct Input; — 0 byte!
  • struct Output; — 0 byte!
  • struct Enabled; — 0 byte!
  • struct Disabled; — 0 byte!

Và struct dùng chúng:

struct Pin<T> { address: u32, _mode: T }

  • size_of::<Pin<Input>>() = 4 bytes (chỉ địa chỉ!)
  • size_of::<Pin<Output>>() = 4 bytes (chỉ địa chỉ!)

Trạng thái không tốn bộ nhớ nào cả! 🎉

⚡ Machine Code Giống Hệt!

🤯 Điều kỳ diệu nhất:

Code dùng Typestate (tiện lợi, an toàn):

let pin = Pin::new(0x48000000).into_output();
pin.set_high();

Và code ghi thẳng vào register (kiểu cũ, C-style):

unsafe { write_volatile(0x48000018 as *mut u32, 0x100); }

→ Tạo ra machine code GIỐNG HỆT nhau! Compiler tối ưu hóa hoàn toàn! 🚀

📊 So Sánh Thực Tế

Tiêu chí C (ghi thẳng) Rust (Typestate)
🏃 Tốc độ Rất nhanh ✅ Rất nhanh (giống hệt)
💾 Bộ nhớ Tối thiểu ✅ Tối thiểu (giống hệt)
🛡️ An toàn ❌ Dễ lỗi ✅ Compiler kiểm tra
📖 Dễ đọc ❌ Khó hiểu số magic ✅ Code rõ ràng
🔧 Bảo trì ❌ Dễ nhầm ✅ Compiler giúp đỡ

🏆 Đây là điểm đặc biệt nhất của Rust!

Ngôn ngữ khác phải chọn một trong hai:

  • 🏃 Nhanh (như C) nhưng không an toàn
  • 🛡️ An toàn (như Python) nhưng chậm

Rust cho bạn cả hai: an toàn + tiện lợi + nhanh = không tốn thêm tài nguyên! 🎯✨

Đó là lý do Rust được dùng trong rocket, OS, trình duyệt web, và cả robot nhỏ! 🚀🤖
