🤹 Concurrency là gì?

🎪 Robot thực tế phải làm rất nhiều việc:

  • 🌡️ Đọc cảm biến mỗi giây
  • 💡 Nhấp nháy đèn LED
  • 📡 Gửi dữ liệu qua UART
  • 🔘 Đọc trạng thái nút bấm
  • ⏱️ Đếm thời gian chính xác

Concurrency là cách robot làm "gần như cùng lúc" — thực ra là chuyển qua chuyển lại rất nhanh! Giống như người biểu diễn tung hứng nhiều bóng! 🎪

🌟 Ví von: Người tung hứng

Người tung hứng giỏi có thể "juggle" 5-7 quả bóng cùng lúc! Thực ra mỗi tay chỉ cầm 1 quả, nhưng chuyển đổi nhanh đến mức trông như làm tất cả cùng lúc. Robot cũng vậy! 🎯

Robot "tung hứng" nhiều việc cùng lúc — chuyển đổi nhanh đến mức trông như làm cùng lúc!

⚠️ Vấn Đề Nguy Hiểm: Data Race!

💡 Giải thích đơn giản trước: Data Race giống như hai bạn cùng viết lên một tờ giấy, mỗi người viết một nửa câu. Nếu ai đó đọc tờ giấy đúng lúc đang viết dở — họ sẽ đọc được câu bị hỏng! Robot cũng vậy khi dữ liệu bị interrupt "chen ngang" giữa chừng.
⚠️ Chuyện gì xảy ra nếu interrupt gõ vai robot giữa chừng?

Hãy tưởng tượng robot đang cộng hai số lớn (ví dụ 16-bit), nhưng chip chỉ tính 8-bit mỗi lần:

  1. 📝 Robot đang ghi số 256 (= 0x0100) vào bộ nhớ
  2. ✍️ Ghi byte cao: 0x01 vào ô đầu — xong!
  3. ⚡ INTERRUPT XẢY RA! Interrupt handler đọc biến này...
  4. 📖 Interrupt đọc được: 0x0100 nhưng byte thấp là 0x00 cũ! → Đọc nhầm giá trị!
  5. ✍️ Robot tiếp tục ghi byte thấp: 0x00...

🚨 Data Race! Dữ liệu bị đọc sai vì interrupt cắt ngang giữa chừng! Rất nguy hiểm! 😱

Data Race: code chính đang viết dữ liệu thì interrupt chen vào đọc giữa chừng — đọc được dữ liệu sai!

🚫 Critical Section — Treo Biển "Không Làm Phiền"

🌟 Ví von: Biển "Không làm phiền" trong khách sạn

Khi bạn đang ngủ, treo biển "Do Not Disturb" — không ai được gõ cửa! Sau khi xong việc, bạn bỏ biển xuống, mọi người có thể vào lại.

Critical section cũng vậy: cortex_m::interrupt::free(|_| { ... }) — trong đó không có interrupt nào được "gõ cửa" robot! 🏨

🦀 Rust là ngôn ngữ DUY NHẤT đảm bảo "no data race" tại compile time!

Các ngôn ngữ khác chỉ phát hiện data race khi chạy (runtime) — đôi khi lỗi chỉ xuất hiện ở sản phẩm của khách hàng! 😱

Rust phát hiện ngay khi bạn đang gõ code — compiler sẽ báo lỗi trước khi bạn kịp chạy! ✅

💡 Quy tắc vàng: Critical section càng ngắn càng tốt! Tắt interrupt lâu → robot có thể bỏ lỡ các tín hiệu quan trọng. Chỉ tắt đủ thời gian để làm việc cần làm, rồi bật lại ngay! ⚡
🤖 Pseudo-code vui vẻ — Critical Section
// 🚫 Treo biển "Không làm phiền":
treo_bien_khong_lam_phien() { // tắt interrupt
  // Trong đây, KHÔNG AI được "gõ vai" robot
  bien_dung_chung = gia_tri_moi; // ghi an toàn!
}
bo_bien_xuong() // bật interrupt lại

// Rust dùng cortex_m::interrupt::free() cho việc này!
