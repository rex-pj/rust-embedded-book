🛠️ Tại sao cần công cụ?

Để xây nhà, thợ cần búa, cưa, máy khoan... Để lập trình robot, bạn cũng cần hộp công cụ đặc biệt! Mỗi công cụ có vai trò riêng, cùng nhau tạo nên robot hoàn chỉnh! 🏗️

Bạn cần 4 công cụ chính: trình biên dịch, quản lý gói, robot ảo và kính lúp debug!

🔨 Các công cụ thiết yếu

🏭
Rust Compiler (rustc)
Nhà máy dịch "ngôn ngữ Rust" thành "ngôn ngữ robot hiểu được" (mã máy) ⚙️
📦
Cargo
Người quản lý dự án — lo tất cả mọi việc: tải thư viện, build, chạy test 📋
🎮
QEMU
Robot ảo! Luyện tập mà không cần robot thật. Như trình giả lập game 🕹️
🔍
GDB / probe-rs
Kính lúp siêu mạnh — nhìn xem robot đang làm gì bên trong, từng bước một 🔬
🔌
OpenOCD / ST-Link
Dây cáp đặc biệt nối máy tính với robot thật để nạp code và debug 🖥️↔️🤖
🍪
cargo-generate
Mẫu dự án sẵn có — như khuôn bánh! Dùng khuôn để tạo project mới nhanh hơn 🎂

🔄 Quy trình làm việc

🌟 Từng bước tạo robot
  1. ✍️ Viết code — Dùng text editor yêu thích (VS Code, nvim...)
  2. 🔨 Build (cargo build) — Cargo + rustc dịch code thành file binary
  3. ⚡ Chạy — Dùng QEMU (giả lập) hoặc flash lên robot thật
  4. 👀 Xem kết quả — Robot làm đúng chưa?
  5. 🐛 Debug — Dùng GDB/probe-rs nếu có vấn đề
  6. 🔁 Lặp lại — Cải thiện và thử lại!

💻 Cài đặt như thế nào?

Bước 1: Cài Rust

Vào rustup.rs và làm theo hướng dẫn — tự động cài tất cả! ✨

Bước 2: Thêm target cho ARM

Robot dùng chip ARM, cần thêm "target" phù hợp:

rustup target add thumbv7em-none-eabihf

Bước 3: Cài QEMU (robot ảo)

Tải từ website QEMU hoặc qua package manager của hệ điều hành

Bước 4: Cài cargo-generate

cargo install cargo-generate

💡 Mẹo cho người mới: Bắt đầu với QEMU trước! Bạn không cần mua robot thật ngay. QEMU giả lập robot hoàn toàn miễn phí trên máy tính của bạn! 🎉
⚠️ Cross-compilation: Máy tính bạn dùng x86/x64, nhưng robot dùng ARM. Cargo + rustc sẽ tự động "dịch sang ngôn ngữ ARM" — gọi là cross-compilation! Phức tạp nhưng Cargo lo tất cả cho bạn. 🎯
